# 思维导图
## 操作介绍 ⭐️⭐️⭐️

换行 ctrl + enter

## 基本操作 ⭐️⭐️⭐️
1. tab 置入子项目
2. enter 置入平级项目
3. ctrl + alt + f  专注模式

思维导图与大纲模式

## 进阶操作 ⭐️⭐️⭐️
1. 联系 ctrl+shift+r
2. 概要 简明扼要 概括要点
3. 外框 ctrl+shift+b 高光内容
4. 笔记 ctrl + shift + n 小图标隐藏内容
5. 标签 外显内容

## 视觉优化 ⭐️⭐️
1. 图标
2. 贴纸

##  如何快速学会一项技能 ⭐️⭐️⭐️
1. 理解基本原理
2. 指定学习地图
3. 刻意练习 不断重复
4. 指定细节 积极创造正反馈


## 制作导图 ⭐️⭐️⭐️
低效导图
1. 抄书型
2. 涂鸦型
3. 迷宫型

找到联系在画导图

高阶导图

1. 分成不同层次，从高向下分析（<font color="#8064a2">同级切割，自上而下</font>）
2. 理解这个层次的全部内容
3. 思考他们之间的逻辑关系，尽量用等分思维来解读
4. 画出导图，并分析结果备注在导图上
