# 2— 核心内容
### 0.1— jdk
#### 0.1.1— JDK下载
官方网址下载:[jdk官网地址](https://www.oracle.com/cn/)
注意事项
1. 针对不同的操作系统，需要下载对应版本的JDK。
2. 如果你的电脑是windows32位的，建议重装系统，重装成64位的操作系统。因为Java从9版本开始，就已经不提供32位版本的安装包了。
#### 0.1.2— JDK安装
傻瓜式安装，下一步即可。默认的安装路径是在C:\Program Files\Java\jdk-24\
#### 0.1.3— 安装目录

| 目录名称    | 说明          |
| ------- | ----------- |
| bin     | 存放各种工具命令    |
| conf    | 存放相关配置文件    |
| include | 存放平台特定头文件   |
| jmods   | 存放各种模块      |
| legal   | 各种模块的授权文档   |
| lib     | 工具的一些补充jar包 |
