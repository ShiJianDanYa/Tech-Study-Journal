### 2.2— 常见命令

| 盘符       | 作用        |
| -------- | --------- |
| dir      | 查看当前路径下内容 |
| 盘符+冒号    | 盘符切换      |
| cd\      | 回退到盘符目录   |
| cd..     | 返回上一级     |
| cd 目录    | 进入单级目录    |
| cd 目录\目录 | 进入多级目录    |
| cls      | 清屏        |
| exit     | 退出命令提示符窗口 |
