# 3— 注意事项
## 1— for和while区别
* 相同点 运行规则都是一样的
* 区别 
	* for循环中，控制循环的变量。因为归属for循环的语法结构中，在for循环结束后，就不能再次被访问到了
	* while循环中，控制循环的变量，对于while循环来说不归属于其语法结构中，在while循环结束后，该变量还可以继续使用
![[循环结构_区别_2026-04-07_10-31-39.png]]
这个说法不绝对 for也可以写在外面啊
![[循环结构_区别_2026-04-07_10-32-26.png]]
真正的区别是使用习惯
for和while区别
1. for循环中：知道循环次数和范围
2. while循环：不知道循环的次数和范围，只知道循环的结束条件
## 2— for和while与do...while特点
* for和while 先判断后执行
* do...while 先执行后判断
