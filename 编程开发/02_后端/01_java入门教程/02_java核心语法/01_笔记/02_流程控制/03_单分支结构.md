# 1— 需要解决问题
如果满足对应条件，则执行对应的代码,否则不执行
# 2— 本质 意义 解决方法
## 1— 单分支判断语句
### 1.1— 判断语句格式
```java
if(关系表达式){
语句体;
}
```

代码演示
```java

import java.util.Scanner;  
  
/**  
 * ClassName: CapacityForLiquor * Package: 课堂代码.ch02分支结构.ch01单分支
```

### 1.2— 执行流程
![[单分支结构_执行流程_2026-04-06_13-42-24.png]]

执行流程：
Ⅰ.首先计算关系表达式的值
Ⅱ.如果关系表达式的值为 true 就执行语句体
Ⅲ.如果关系表达式的值为 false 就不执行语句体
Ⅳ.继续执行后面的其他语句
# 3— 注意事项
if判断语句 细节
1. if语句大括号的省略 如果大括号中语句体只有一行，大括号可以省略(个人建议大括号不要省略)
2. 小括号后面不能有分号，这样会拆开if的语句结构
3. if语句大括号的位置 左括号写在上一行的末尾，不要单独写一行
4. 判断布尔类型的变量，直接把变量写在小括号中即可

```java
package 课堂代码.ch02分支结构.ch01单分支;  
  
/**  
 * ClassName: 注意事项  
 * Package: 课堂代码.ch02分支结构.ch01单分支  
 * Description:  
 * * @Author:锏澹  
 * @Create: 2026/4/20 - 20:01  
 * @Version: v1.0  
 * */public class 注意事项 {  
    static void main(String[] args) {  
       //if 注意事项  
       //1.大括号的开头可以另起一行书写，但是建议写在第一行的末尾  
       //2.小括号后面不能有分号，这样会拆开if的语句结构  
       //3.在语句体中，如果只有一句代码，大括号可以省略不写。个人建议，大括号还是不要省略  
       //4.如果对一个布尔类型的变量进行判断，不要用==号  
  
       //大括号的开头可以另起一行书写，但是建议写在第一行的末尾  
       if(true)  
       {  
          System.out.println("hello world");  
       }  
       //小括号后面不能有分号，这样会拆开if的语句结构  
       if(false);{  
          System.out.println("hello world");  
       }  
       //在语句体中，如果只有一句代码，大括号可以省略不写。个人建议，大括号还是不要省略  
       if(true)  
          System.out.println("hello world");  
  
       //如果对一个布尔类型的变量进行判断，不要用==号  
       boolean b = true;  
       if(b){  
          System.out.println("hello world");  
       }  
    }  
}
```

# 1— 知识扩展
## 1— 紧凑风和折叠风
K&R风格（紧凑风）：左括号在上一行的末尾
Allman风格（折叠风）：左括号另起一行

K&R风格：
创造c语言的两位大神
布莱恩·克尼汉BrianKernighan
丹尼斯·里奇 Dennis Ritchie
《The CProgramming Language》 书里面两种风格都用到了
仿照大神的风格，从两个名字各取一个字母，左括号在上一行的末尾-->K&R风格
Java Go
Java：Oracle官方示例 谷歌编码规范 Spring框架-->K&R风格

Allman风格：
BSD UNIX操作系统的主要开发者艾瑞克·奥尔曼（Eric·Allman）C#-->A1lman风格

## 2— 小括号后面不能有分号

```java
if(true);{
System.out.println("你好啊世界");
}
```
最终无论 （）内的是否成立
你好啊世界 语句都会输出 

因为Java把代码理解为
```java
//分号;判断---语句体之间的联系切断
if(true);
//独立的代码快，不受上面的if控制
System.out.println("语音警告，当前体温已经超过了38度---1");

```

## 3— 布尔类型的变量
布尔类型的变量 判断布尔类型的变量，为什么直接把变量写在小括号中
```java
boolean b = false;

//==：判断
//=：辅助
if(b =true){
System.out.println("为真");
}
```

因为要避免 b=true 情况出现
这样无论 b 原值无论如何 语句都会执行
