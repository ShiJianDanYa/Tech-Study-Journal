# 1— 需要解决问题
如何让程序一直开着，等待操作
# 2— 本质 意义 解决方法
永远无法终止的循环

格式分别如下
for
```java
for(;;){
	语句体;
}
```

while
```java
while(true){
	语句体;
}
```
do...while
```java
do{
	语句体;
}while(true);
```
无限循环是不知道次数的
