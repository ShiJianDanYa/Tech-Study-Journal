# 1— 需要解决问题
已经确定好循环次数，该怎么循环？
# 2— 本质 意义 解决方法
## 1— for循环
### 1.1— 格式
```java
for(初始化语句;条件判断语句;条件控制语句){
	循环体语句;
}
```
以100 为例
```java
for(int i = 1;i<=100;i++){
	循环体语句;
}
```

![[循环结构_for语句_2026-04-06_18-15-53.png]]
### 1.2— 执行流程
图示 解读 
![[循环结构_for语句_2026-04-06_18-17-34.png]]

1. 执行初始化语句
2. 执行条件判断语句，看其结果是否成立
	1. 成立：执行循环体语句
	2. 不成立：结束循环
3. 执行条件控制语句
4. 回到2继续执行条件判断语句

### 1.3— 流程图
![[循环结构_for语句_2026-04-06_18-20-23.png]]

### 1.4— 循环需要注意的小细节

1. 初始化语句只执行一次
2. 判断语句成立，循环继续
3. 判断语句不成立，循环结束
