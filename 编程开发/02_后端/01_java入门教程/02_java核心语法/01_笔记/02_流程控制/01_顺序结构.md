# 知识目录

# 1— 引入问题
java 程序的默认执行顺序是什么?
# 2— 核心内容
顺序结构
是 java 的默认的执行流程,
按照代码的先后顺序,
从上到下依次执行.

代码
```java
/**  
 * ClassName: order * Package: 课堂代码.顺序结构  
 * Description:  
 * * @Author:锏澹  
 * @Create: 2026/4/4 - 17:55  
 * @Version: v1.0  
 * */public class order {  
        public static void main(String[] args) {  
                System.out.println("执行顺序1");  
                System.out.println("执行顺序2");  
                System.out.println("执行顺序3");  
                System.out.println("执行顺序4");  
                System.out.println("执行顺序5");  
                System.out.println("执行顺序6");  
        }  
}
```
