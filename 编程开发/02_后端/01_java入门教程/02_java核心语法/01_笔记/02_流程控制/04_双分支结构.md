# 1— 需要解决问题
满足某些条件，就执行特定代码，否则就执行另一段代码
# 2— 本质 意义 解决方法
## 1— 双分支判断语句
### 1.1— 双分支判断语句格式
```java
if(关系表达式){
	语句体A;
}else{
	语句体B;
}
```
### 1.2— 执行流程
![[双分支结构_执行流程_2026-04-06_13-49-38.png]]

执行流程：
Ⅰ.首先计算关系表达式的值
Ⅱ.如果关系表达式的值为 true 就执行语句体 1 
Ⅲ.如果关系表达式的值为 false 就执行语句体 2 
Ⅳ.继续执行后面的其他语句
