# 1— 需要解决问题
控制 无限循环
# 2— 本质 意义 解决方法
## 1— 循环控制语句
### 1.1— 关键字 break
**不能单独书写**，只能在<font color="#ff0000">switch</font>或<font color="#ff0000">循环</font>中使用，表示结束跳出的意思
(⚠️ switch 是选择语句可不是循环别搞错了)
![[流程控制_循环控制_2026-04-07_09-21-33.png]]
格式如下
```java
for (int i = 0;i< 100;i++) {
if(i == 6){
	break;
	}
System.out.println("循环第"+i+"次");
}
```

### 1.2— 关键字 continue
**不能单独书写**，只能写在<font color="#ff0000">循环</font>中,结束本次循环，继续下次循环
![[流程控制_循环控制_2026-04-07_10-11-02.png]]
格式如下
```java
for(int i = 0;i< 100;i++) {
if(i == 6){
	continue;
}
System.out.println（"循环第"+i+"次"）；
```

# 3— 注意事项
## 1— break 跳出的循环
默认情况下，`break`语句**只能跳出它所在的那一层循环**，无法直接跳出多层嵌套循环
