# 1— 需要解决问题
一件事有多种情况，只执行其中满足条件的语句
# 2— 本质 意义 解决方法
## 1— 多分支结构
### 1.1— if多分支结构
多分支<font color="#d83931">判断</font>语句
#### 1.1.1— 格式
```java
if(关系表达式A){
	语句体A;
} else if(关系表达式B){
	语句体B;
} else if（关系表达式C）{
	语句体C;
}
...
else {
语句体 N;
}
```
#### 1.1.2— 流程
![[if多分支语句执行流程_2026-04-21_14-29-24.png]]

从上往下依次进行判断
只有要有一个判断为真，就执行对应的语句体如果所有的判断都为假，就执行 else 的语句体

#### 1.1.3— 执行流程:
Ⅰ.首先计算<font color="#d83931">关系表达式 1</font> 的值
Ⅱ.如果为 true 就执行语句体 A；如果为 false 就计算<font color="#d83931">关系表达式 2</font> 的值
Ⅲ.如果为 true 就执行语句体 B；如果为 false 就计算<font color="#d83931">关系表达式 3</font>
的值
Ⅳ. …··
Ⅴ.如果所以关系表达式结果都为 false，就执行语句体 N+1。
### 1.2— switch多分支结构
多分支<font color="#d83931">选择</font>语句,把所有的选择一一列举出来，根据不同的条件<font color="#d83931">任选其一</font>
#### 1.2.1— 格式
```java
switch(week){
	case 值1:
		语句体1;
		break;
	case 值2:
		语句体2;
		break;
	...
	default:
		语句体n;
		break;
}
```
这里的<font color="#d83931">week</font>是一个表达式(是一个<font color="#d83931">具体数值</font>),不是一个判断
#### 1.2.2— 流程图
![[多分支结构_switch执行流程_2026-04-06_14-25-56.png]]

#### 1.2.3— 执行流程
Ⅰ.首先计算表达式的值。
Ⅱ.依次和 case 后面的值进行比较，如果有对应的值，就会执行相应的语句，在执行的过程中，遇到 break 就会结束。
Ⅲ.如果所有的 case 后面的值和表达式的值都不匹配，就会执行 default 里面的语句体，然后结束整个 switch 语句。
# 3— 注意事项
## 1— switch注意事项
1. 表达式：结果（字符/整数 byte short int/<font color="#d83931">枚举(jdk 5)</font>/<font color="#d83931">字符串(jdk 7)</font>）
2. case：被匹配的值，只能是真实的数据(被匹配的值 被复制后的变量也不可以)
3. case：值不允许重复
4. break：表示中断，结束的意思，结束 switch 语句
5. default：所有情况都不匹配，执行该处的内容

## 2— default 的位置和省略
位置 case 和 default 是没有标准的的上下之分的，位置可以任意的书写
为了方便观看，提高代码的阅读性
一般来讲，case 是从小到大书写的，default 是写在最下面的。
default 也可以省略不写，在此时如果所有的 case 都不匹配，则没有任何输出结果

## 3— case 穿透
在我们书写代码的时候，如果 break 没有写，此时就会触发 case 穿透问题。

执行流程
1. 拿着小括号中的表达式的值跟下面的 case 进行匹配
2. (正常情况）如果匹配符合，就会执行 case 里面的语句体，遇到 break 结束整个 switch
3. (非正常情况) 如果执行语句体的时候没有遇到 break，那么程序会继续执行下一个 case 的语句体，直到遇到 break 或者运行完整的 switch 为止

应用场景：
当多个 case 的语句体重复的时候，利用 case 穿透可以节省代码。

## 4— switch 的新特性
注意 新特性 
jdk 12 推出预览版
jdk 14 正式版
运行前提 jdk14 及其以上
### 4.1— 箭头标签
源代码
```java
switch(number)
{
	case 1:
		System.out.println("一");
	break;
	
	case 2:
		System.out.println("二");
	break;
	
	case 3:
		System.out.println("三");
	break;
	
	default:
		System.out.println("没有这个星期");
	break;
}
	
```
通过箭头简化后
```java
switch(number)
{
	case 1 ->{
		System.out.println("一");
	}

	case 2->{
		System.out.println("二");
	}
	
	case 3->{
		System.out.println("三");
	}
	
	default->{
		System.out.println("没有这个星期");
	}
	
}
```
简化后 自带 break;
只有一个语句{ } 也可以省略
最终简化版本
```java
switch(number){
case 1 ->System.out.println("一");
case 2->System.out.println("二");
case 3->System.out.println("三");
default->System.out.println("没有这个星期");
}
```
### 4.2— case 后面可以写多个值
条件不同，但是执行语句相同，那么就可以在 case 后面写多个值,用逗号隔开。
```java
switch(number){  
    case 1,2,3,4,5 ->System.out.println("工作日");  
        default->System.out.println("没有这个星期");  
}
```
### 4.3— switch 可以有运行结果
我们如果后面需要用到 switch 运行结果呢？
```java
//需要一个变量接受数据  
String name =switch(number){  
    case 1,2,3,4,5->{  
        yield"工作日"; 
    }  
    default->{  
        yield"没有这个星期";  
    }  
//大括号后面还需要补一个分号  
};
```
还是太复杂了能不能在简化一下
```java
//需要一个变量接受数据  
String name =switch(number){  
    case 1,2,3,4,5 ->"工作日";  
        default->"没有这个星期";  
//大括号后面还需要补一个分号  
};
```
### 4.4— yield 关键字
`yield` 用于在 `switch` 表达式中显式返回值
## 5— switch 的和 if 第三种格式各自的使用场景
if 语句：对范围进行判断
switch 语句：case 的值是有限的
