# 1— 需要解决问题
至少执行一次的循环，先执行再判断的循环，这种情况怎么办？
# 2— 本质 意义 解决方法
## 1— do...while 循环语法格式
```java
初始化语句;
do{
	循环体语句;
	条件控制语句;
}while(条件判断语句);

```
