# 2— 本质 意义 解决方法
## 1— 二维数组
### 1.1— 二维定义
数组中的每一个元素都是一个一维数组,这个数组就是一维数组.(就是数组里面包数组)
### 1.2— 初始化
#### 1.2.1— 静态初始化
格式
完整
```java
// 数据类型[][] 数组名 = new 数据类型[][]{元素1,元素2,元素3,......}
int[][] arr = new int[][]{{12,24,36},{666,888},{10,20,30},{999}};
```
简写
```java
// 数据类型[][] 数组名 = {元素1,元素2,元素3,......}
int[][] arr = {{12,24,36},{666,888},{10,20,30},{999}};
```
#### 1.2.2— 动态初始化
格式
```java
// 数据类型[][] 数组名 = new 数据类型[长度1][长度2]
int[][] arr = new int[3][5];
```
动态化数组 arr 也是有数据的
![[数组_二维数组_2026-04-08_11-27-58.png]]
数组操作 演示代码
```java
public class TwoDimensionalArray {  
    public static void main(String[] args){  
        /*  
        第一排：张无忌 赵敏 周芷若  
        第二排：张三丰 宋远桥 殷梨亭        
        第三排：灭绝 陈昆 玄冥二老 金毛狮王
        第四排：杨逍 纪晓芙  
        */       
        //二维数组静态初始化  
        String[][] names = new String[][]{  
                {"张无忌","赵敏","周芷若"},  
                {"张三丰","宋远桥","殷梨亭"},  
                {"灭绝","陈昆","玄冥二老"},  
                {"杨逍","纪晓芙",""}  
        };
```

### 1.3— 二维数组数据的访问
#### 1.3.1— 行索引
格式
```java
//二维数组的访问  
//数组名[行索引]  
String[] arr3 = names[2];
System.out.println("直接输出:"+arr3);  
//遍历数组  
for(int i = 0;i < arr3.length;i++){  
    System.out.println(arr3[i]);  
}
```
结果
![[数组_二维数组_2026-04-08_14-42-47.png]]
直接输出的结果是 存储数组的内存地址
遍历输出的是下标为2的数组的所有元素
#### 1.3.2— 列索引
格式
```java
//数组名[行索引][列索引]  
//输入的和你实际上认为的是相差1,可以理解为+1  
String name = names[0][1];  
System.out.println(name);
```
结果
![[数组_二维数组_2026-04-08_14-43-16.png]]
数组下表为0的第2个(下表为1)的元素
### 1.4— 二维数组的长度
格式
```java
//数组的长度 数组名.length  
//直接二维数组.length显示数组的行数
System.out.println(names.length);  
//二维数组.length[行数]显示数组的此行的列数
System.out.println(names[0].length);  

```

### 1.5— 遍历二维数组
格式
```java
//遍历二维数组(静态初始化不规则) 
// names.length需要遍历的行数
for(int i = 0;i < names.length;i++){  
//names[i].length需要遍历此行的列数
    for(int j =0;j< names[i].length;j++){  
        System.out.print(names[i][j]+"\t");  
    }  
    System.out.println();  
}
```
