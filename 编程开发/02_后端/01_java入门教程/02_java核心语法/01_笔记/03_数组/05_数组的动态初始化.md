# 1— 需要解决问题
创建数组的时候只知道数组元素的数量和类型，不知道数组元素的具体值
# 2— 本质 意义 解决方法
## 1— 动态初始化
动态：在定义变量，数组，对象的时候，数据是不**确定**的
初始化：创建数组的时候，直接给数组赋值

不知道数据如何赋值（不知道数据具体的值 1啊还是1000但是知道其数据类型）

动态初始化：创建数组的时候指定长度，由系统为数组分配初始值
### 1.1— 动态初始化默认初始化值
| 数据类型 | 默认值    |
| :--- | :----- |
| 整型   | 0      |
| 浮点数  | 0.0    |
| 字符   | \u0000 |
| 布尔   | false  |
| 引用   | null   |

## 2— 格式
格式
```java
//数据类型 数组名[] = new 数据类型[长度];
int arr [] = new int[3];
```
代码演示
```java
public class Dynamic_initialization {  
    public static void main(String[] args) {  
        /*需求：键盘录入5个的整数，存入数组当中，并进行遍历  
        * 动态初始化的格式：  
        * 数据类型[] 数组名= new 数据类型[数组的长度];  
        * */        
        int [] ageArr = new int[5];  
        System.out.println("请输入5个整数");  
        Scanner sc = new Scanner(System.in);  
        for (int i = 0; i < ageArr.length; i++) {  
            System.out.println("请输入第"+(i+1)+"个整数");  
            ageArr[i] = sc.nextInt();  
        }  
        //一个循环只做一件事情  
        //遍历数组  
        for (int i = 0; i < ageArr.length; i++) {  
            System.out.println(ageArr[i]);  
        }  
    }  
}
```

# 3— 注意事项

对于数组的动态初始化，没有简化 ，只有推导
# 1— 知识扩展
数组动态初始化 可以用 int[] arr = [5] 这种方式简写吗
### 0.1— 为什么 `int[] arr = [5]` 不行？

- Java 的语法规则中，数组初始化必须显式使用 `new 类型[]`（动态 / 静态）或 `{}`（简化静态初始化），单独的 `[5]` 不是合法的数组字面量；
- 这种 `[5]` 的写法更接近 JavaScript 等脚本语言的数组创建方式，但 Java 作为强类型语言，有严格的语法规范，不支持这种简写。

### 0.2— 总结

1. Java 中**不支持** `int[] arr = [5]` 这种简写，这是语法错误；
2. 创建包含元素`5`的 int 数组，推荐用 `int[] arr = {5}` 或 `int[] arr = new int[]{5}`；
3. 动态初始化的核心是 “先指定长度，后赋值”，写法为 `int[] arr = new int[长度]`，再通过下标赋值。
