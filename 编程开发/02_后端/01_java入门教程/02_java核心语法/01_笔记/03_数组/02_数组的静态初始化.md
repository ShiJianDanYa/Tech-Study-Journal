# 1— 需要解决问题
知道具体数组元素情况下，如何创建数组
# 2— 本质 意义 解决方法
## 1— 静态初始化
初始化：是指在定义变量，数组，对象的时候进行赋值
简单：初始化 定义和赋值同时进行
```java

//初始化变量number
int number = 10;

```

静态：在定义变量，数组，对象的时候，数据是静止的，确定的
简单：静态 定义时数据是确定的

数组的静态初始化：创建数组的时候直接给数组元素赋确定的字面量值
简单：数组初始化 定义数组的同时给其赋确定的值
## 2— 静态初始化格式
完整格式
```java

// 数据类型 数组名 [] = new 数据类型 []{数据值，数据值...}；
int arr [] = new int [] {1,2,3};

```
简写格式
```java
// 数据类型 数组名 = {数据值，数据值...}
int arr [] = {1,2,3}
```

代码演示
```java
package ch01_课堂代码.ch01_数组的静态初始化;  
  
  
/*  
    按照要求定义数组，数组中的数据任意书写  
    1.定义数组存储3位同学的年龄     整数  
    2.定义数组存储5位同学的身高     小数  
    3.定义数组存储3位同学的名字     字符串  
  
        静态初始化            完整格式：                数据类型[] 数组名= new 数据类型[]{元素1,元素2,元素3,...，元素n}；  
  
            简写格式：                数据类型[] 数组名 = {元素1，元素2，元素3，.，元素n}；  
*/  
public class ch01_数组静态初始化 {  
    public static void main(String [] args){  
        //1. 定义数组存储3位同学的年龄  
        int[] ageArr1 = new int[] {19,20,25};                          //整数  
        int[] ageArr2 = {19,20,25};  
        //2. 定义数组存储5位同学的身高  
        double[] heightArr1 = new double[]{1.75,1.80,1.65,1.70,1.85};  //小数  
        double[] heightArr2 = {1.75,1.80,1.65,1.70,1.85};  
        //3.定义数组存储3位同学的名字  
        String[] nameArr1 = new String[] {"张三","李四","王五"};        //字符串  
        String[] nameArr2 = {"张三","李四","王五"};  
    }  
}
```
# 1— 注意事项
## 1— 静态初始化注意事项
1. 数组是一个连续的空间
2. 数组一旦定义完毕长度是不可变
