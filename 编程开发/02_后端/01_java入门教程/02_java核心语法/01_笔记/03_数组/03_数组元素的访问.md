如何使用数组中的特定元素
# 2— 本质 意义 解决方法
## 1— 访问数组元素
解决问题 如何访问数组中的某一元素
## 2— 索引
### 2.1— 索引定义
为了方便对数组元素的操作，给每个元素都赋予了对应的下标
这就叫作索引
简单：索引  就是 下标
索引可以理解为 数组中存储数据的小格子的编号
![[数组_访问数组_2026-04-07_16-12-20.png]]
索引就是数组的一个编号，也叫作：角标，下标，编号
### 2.2— 索引特点
特点 ：从0开始，连续+1,中间不间断
![[附件存储/16ee4e1af9da3ff59cd19fd95f6232f5_MD5.png]]
注意 只是各自的编号，与里面实际数据无关
## 3— 访问数组元素
### 3.1— 获取
```java
//变量 = 数组名[索引];
int num = arr[5];
```
### 3.2— 修改
```java
//数组名[索引] = 数据值;
arr[5] = 10;
```
### 3.3— 代码演示
代码演示
```java
package ch01_课堂代码.ch02_数组元素的访问;  
  
/**  
 * ClassName: Array_element_access * Package: 数组的练习.数组的访问  
 * Description:  
 * * @Author:锏澹  
 * @Create: 2026/3/2 - 11:32  
 * @Version: v1.0  
 * */public class ch01_数组元素的访问 {  
    public static void main(String[] args) {  
        /*数组中元素的访问：  
        1.获取数据  
        2.修改数据*/  
        //利用静态初始化创建数组  
        int [] ageArr = {19,20,25};  
        //获取数组中的元素  
        //从索引获取，索引特点：索引从0开始，连续加一，中间不间断  
        //获取索引0的数据  
        System.out.println(ageArr[0]);  
        //也可以通过变量  
        int age = ageArr[0];  
        System.out.println(age);  
        //修改数据  
        //格式 ：arr[索引] = 修改的数据  
        //注意 ：一旦修改完毕，原来数据就被覆盖了  
        ageArr [0] = 18;  
        System.out.println(ageArr[0]);  
    }  
}
```
