# 1— 引入问题
需要多个功能相同，但是参数列表不同的方法，用不同的方法名太麻烦了
# 2— 核心思想
## 1— 方法的重载
同一个类中，定义了多个同名的方法，这些方法具有类似的功能
每个方法具有不同的参数类型和参数个数，这些同名的方法，就构成了重载关系
简单理解：同一个类，方法名相同，参数不同的方法，它们之间构成重载,和 `返回值` 无关。
参数不同的情况
1. 个数不同
2. 类型不同
3. 顺序不同
