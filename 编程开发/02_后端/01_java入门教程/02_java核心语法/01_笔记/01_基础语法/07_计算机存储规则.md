#### 3.2.1— 数字存储
##### 3.2.1.1— 二进制
二进制  逢二进一 借一当二 由0和1组成
##### 3.2.1.2— 进制和其对应格式  
此特性在 jdk7 版本添加
* 二进制:由0和1组成,代码中以<font color="#ff0000">0b</font>开头
* 八进制:由 0~7 组成,代码以<font color="#ff0000">0</font>开头
* 十进制:由0~9组成,前面不加任何前缀
* 十六进制:由0~9还有a~f组成,代码中以<font color="#ff0000">0x</font>开头
注意 `区分 字母 o 和数字 0`
##### 3.2.1.3— 为什么用二进制来存储数据
简单 只需要两种状态就可以表示

三原色
1. 计算机中的颜色采用光学三原色
2. 分别为:<font color="#d83931">红</font>,<font color="#00b050">绿</font>,<font color="#0070c0">蓝</font>.也称为<font color="#d83931">RGB</font>
3. 可以写成十进制.(<font color="#d83931">255</font>.<font color="#00b050">255</font>.<font color="#0070c0">255</font>)
4. 也可以写成十六进制(<font color="#d83931">FF</font><font color="#00b050">FF</font><font color="#0070c0">FF</font>)

图片的每一个像素点都是由不同算三原色组成
