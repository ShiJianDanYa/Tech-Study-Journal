## 2— 运算符分类
### 2.1— 算数运算符
➕ ➖ ✖️   除号（/） 取模（%）
注意 `%是取模`, `/才是除号`
#### 2.1.1— 注意事项
1. ➕ ➖ ✖️ / 运算中 有小数参与会可能导致，运算结果不精确
2. 除法中纯整数运算，结果只保留整数部分，舍弃小数部分

 代码演示
```java
package 课堂.运算符.算术运算;  
  
/**  
 * ClassName: sign * Package: 课堂.运算符.java06  
 * Description: * * @Author 锏箪  
 * @Create 2025/3/2719:42  
 * @Version1.0  
 */  
public class sign {  
    public static void main(String[] args) {  
        //运算中有小数参与会可能导致，运算结果不精确  
        System.out.println(3.2+2.001);  
        System.out.println(3.2-2.001);  
        System.out.println(3.2*2.001);  
        System.out.println("-----------------------------------------------------------------------");  
        //➗ 纯整数参与运算，结果只保留整数部分，舍弃小数部分  
        System.out.println(11/2);  
        System.out.println(10/3);  
        System.out.println(11.0/2);  
        //取余 除法运算的余数  
        System.out.println(11%2);  
    }  
}
```

##### 2.1.2.1— 数字相加
数字进行运算时，数据类型不一样，不能直接运算，需要转换。
###### 2.1.2.1.1— 隐式转换和强制转换

 隐式转换（自动类型提升）
<font color="#c00000">取值范围小的转换为取值范围大的</font>
![[运算符_隐式转换_2026-04-03_15-24-48.png|840]]
转换是范围 `由小转大`。
<font color="#c00000">规则</font>
1. 范围小的会自动提升到大的在进行计算
2. byte short <font color="#00b050">char</font> 三种类型 会<font color="#00b050">自动转化为int在进行计算</font>

强制转换
 java中不允许直接将取值大的赋值给取值小的，需要强制转换

<font color="#c00000">格式</font> 
目标数据类型 变量名 = <font color="#00b0f0">（目标数据类型）</font>被作用数据
代码演示
```java
package com.itheima.arithmeticoperator;

public class ArithmeticoperatorDemo3 {
	public static void main(String[] args) {
	byte b1 = 10;
	byte b2 = 20;
	//现在我们要强转的是谁？
	//b1+b2计算之后的结果。
	//(byte)b1+b2强转的是b1，并不是最终的结果
	//b1+b2结果不应该是byte类型吗 不对
	//byte short char 三种类型 会>自动转化为int在进行计算 所以 结果是int类型
	byte result = (byte)(b1 + b2);
	System.out.println(result);//30
```

### 2.2— 自增自减运算符

| 符号  | 作用  | 说明    |
| --- | --- | ----- |
| ++  | 加   | 变量值+1 |
| --  | 减   | 变量值-1 |

#### 2.2.1— 注意事项
++ -- 可以在变量前面也可以在后面
1. 两者在单独一行没有区别
2. ++--在前 先自增自减，再参与运算
3. ++--在后 先参与运算，再自增自减

代码
```java
package ch12运算符.课堂代码.ch02自增自减运算符;  
  
/**  
 * ClassName: auto * Package: 课堂.运算符.java07  
 * Description: * * @Author 锏箪  
 * @Create 2025/3/2721:38  
 * @Version1.0  
 */  
  
public class 自增自减运算符 {  
    public static void main(String[] args) {  
        //自增自减 两者在单独一行没有区别  
        // 但是在表达式中，自增自减在前面，先自增自减，再参与运算  
        //自增自减在后面，先参与运算，再自增自减  
    int a = 10;  
    ++a;  
    System.out.println(a);  
    a = 10;  
    a++;  
    System.out.println(a);  
    a = 10;  
    a--;  
    System.out.println(a);  
    a = 10;  
    --a;  
    System.out.println(a);  
    System.out.println("=============================");  
    //++--在前 先自增自减，再参与运算  
    int w = 10;  
    int b = ++w;  
    System.out.println(w);  
    System.out.println(b);  
    //++--在后 先参与运算，再自增自减  
    int c = w++;  
    System.out.println(w);  
    System.out.println(c);  
    }  
}
```

#### 2.3.2— 赋值运算符注意事项

### 2.6— 短路逻辑运算符
为了提高程序的运行效率

| 符号   | 作用  | 说明              |
| ---- | --- | --------------- |
| &&   | 短路与 | 结果和&相同，但是有短路效果  |
| \|\| | 短路或 | 结果和\|相同，但是有短路效果 |

#### 2.6.1— 短路
<font color="#c00000">左边的表达式能够确定运行结果，右边就不会执行</font>
逻辑短路运算符
```java
package ch12运算符.课堂代码.ch06逻辑短路运算符;  
  
/**  
 * ClassName: 逻辑短路运算符  
 * Package: ch12运算符.课堂代码.ch06逻辑短路运算符  
 * Description:  
 * * @Author:锏澹  
 * @Create: 2026/4/18 - 20:43  
 * @Version: v1.0  
 * */public class 逻辑短路运算符 {  
    static void main(String[] args) {  
       //1. &&  
       //运行结果跟单个&是一样的  
       // 表示两边都为真，结果才是真  
       System.out.println(true && true);//true  
       System.out.println(false && false);//false  
       System.out.println(false && true);//false  
       System.out.println(true && false);//false  
       //2.||       //运行结果跟单个|是一样的  
       //表示两边都为假，结果才是假  
       System.out.println(true || true);// true  
       System.out.println(false || false);//false  
       System.out.println(false || true);//true  
       System.out.println(true || false);//true  
       //3.短路逻辑运算符具有短路效果  
       //简单理解：当左边的表达式能确定最终的结果，那么右边就不会参与运行了  
       int a =10;  
       int b =10;  
       boolean result = ++a < 5 && ++b< 5;  
       System.out.println(result);//false  
       System.out.println(a);//11  
       System.out.println(b);//10  
    }  
}
```

#### 2.6.2— 注意事项
1. &和|,无论左边是true,还是false,右边都要执行
2. &&和||在左边能够确定整个表达式的值的情况下,右边不执行
	1. &&左边为假,则整个表达式为假,右边不执行,整个表达式的结果一定是假
	2. ||左边为真,则整个表达式为真,右边不执行，整个表达式的结果一定是真
	3. 这两种情况下，右边不执行，提高了效率。
3. 最常用的逻辑运算符：&&，||，！
