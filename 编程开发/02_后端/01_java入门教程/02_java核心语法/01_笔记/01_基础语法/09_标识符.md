## 1— 需要解决问题
程序中变量 方法 和类该如何命名
## 2— 本质 意义 解决方法
### 2.1— 标识符的本质
![[标识符_2026-04-02_15-47-15.png]]
是给 <font color="#ff0000">类</font> <font color="#ff0000">方法</font> <font color="#ff0000">变量</font> 起的<font color="#ff0000">名字</font>
### 2.2— 命名规范
#### 2.2.1— 硬性规则（强制要求）
1. 由数字 字母 下划线_ 和美元符号$ 组成
2. 不能以数字开头
3. 不能是关键字
4. 严格区分大小写
#### 2.2.2— 公共建议（软性建议）
1. <font color="#ff0000">方法 变量</font> 小驼峰 除首单词外，其余单词首字母全部大写
2. <font color="#ff0000">类名</font> 大驼峰 单词首字母大写
3. 命名要 <font color="#ff0000">见名知意</font>
