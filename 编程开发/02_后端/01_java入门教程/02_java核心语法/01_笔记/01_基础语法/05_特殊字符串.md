## 1— 需要解决问题
输出字符利用空格对齐，有时会出现怎么也对不齐
## 2— 本质 意义 解决方法
### 2.1— 制表符
\t 制表符
作用 将前面字符串长度<font color="#ff5c00">补齐到8</font> ，或者<font color="#ff5c00">8的整数倍</font> 最少为一空格，最多为8空格
自动补齐 长度 便于后面内容对齐 只会针对前面字符串  

简单理解：长度可变的大空格，打印表格类数据的时候，可以让上下对其。（99 乘法表）
真正的含义：
在前面的字符后面补 1-4 个空格（没有 0 所以不会补 0个空格），让这个整体的长度凑成 4 的整数倍  -`idea`
在前面的字符后面补 1-8 个空格，让这个整体的长度凑成 8 的整数倍

## 3— 注意事项
<font color="#d83931"> cmd输出是补8的倍数，不同编译器补的倍数是不同，需要以实际情况为准 比如idea是4 </font>
代码演示
```java
package ch04字面量;  
  
/**  
 * ClassName: tabs * Package: ch04字面量  
 * Description:  
 * * @Author:锏澹  
 * @Create: 2026/4/15 - 08:27  
 * @Version: v1.0  
 * */public class tabs {  
    static void main(String[] args) {  
       System.out.println("123456789");  
       System.out.println("a"+'\t'+'|');  
       System.out.println("ab"+'\t'+'|');  
       System.out.println("abc"+'\t'+'|');  
       System.out.println("abcd"+'\t'+'|');  
       System.out.println("abcde"+'\t'+'|');  
       System.out.println("abcdef"+'\t'+'|');  
       System.out.println("abcdefg"+'\t'+'|');  
       System.out.println("abcdefgh"+'\t'+'|');  
    }  
}
```
