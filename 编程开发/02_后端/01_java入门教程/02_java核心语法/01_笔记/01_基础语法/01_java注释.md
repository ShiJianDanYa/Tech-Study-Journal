### 4.2— 多行注释为什么不支持嵌套

```java
/*输出一段文本，/*内容为：HelloWorld输出一段文本，*/内容为:HelloWorld*/
System.out.println("Hello World");
```

因为注释的逻辑 是从 `/*`  开始注释 寻找到 `*/ ` 才是结束注释
但是 如果 这个注释里有 ` /*` 只会把他当作注释内容
不会当作注释开头，那另一边的 `*/` 就会,变成一个没有 `/*` 的注释
自然会报错
