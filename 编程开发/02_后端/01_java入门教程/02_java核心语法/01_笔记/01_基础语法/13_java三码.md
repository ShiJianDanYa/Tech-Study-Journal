# 三码
原码、反码、补码
* 原码：十进制数据的二进制表现形式，最左边是符号位，0为正，1为负
* 反码：正数的补码反码是其本身，负数的反码是符号位保持不变，其余位取反
* <span style="background:#ff4d4f">补码：正数的补码是其本身，负数的补码是在其反码的基础上+1</span>
三码中正数的三码是不变的，变得只有负数
