## 1— idea介绍
### 1.1— IDEA的基本使用
IntelliJIDEA是一款强大的 Java集成开发环境（IDE），被广泛认为是开发Java 程序的最佳工具之一。
在开始本文档之前，请确保你已正确安装JDK 和 IDEA
### 1.2— 项目层级结构
在IDEA中，Java项目按以下层级进行组织：
1. Project（项目）：整体工程，比如微信、QQ等应用。
2. Module（模块）：项目的功能子模块，如聊天、通讯录等。
3. Package（包）：模块内的业务分类，例如聊天模块中的朋友圈、视频号等。
4. Class（类）：具体的代码单元，实际开发工作主要在类中完成。

注：模块暂时用不到，我们先不创建，等后面章节用到了模块，我们再进行使用
目前使用的层级：项目(Project）→包(Package)→类(Class).

## 2— 使用 IDEA创建第一个Java程序
### 2.1— 打开IDEA
1. 双击IDEA图标启动软件。
2. 确保进入主界面后继续下一步操作。

![[idea_2026-04-02_16-39-28.png]]
### 2.2— 新建JaVa项目结构
创建项目需要遵循以下层级：
1. 新建项目(Project）。
2. 新建包（Package）。
3. 新建类（Class）。

#### 2.2.1— 新建项目
1. 点击主界面中的 NewProject。![[idea_2026-04-02_16-49-31.png]]
2. 在弹出的窗口中：
	1. 选择项目类型：选择Java 项目。
	2. 填写项目名称和路径：
		1. Name(名称)：输入项目名称。
		2. Location(位置)：选择项目存储路径（点击右侧目录图标可修改存储位置）。
		3. addsamplecode(添加示例代码)：取消添加简单代码选项
	3. 点击Create创建项目。![[idea_2026-04-02_16-53-09.png]]

#### 2.2.2— 新建包
1. 在项目资源管理器中，右键展开模块下的src文件夹。
2. 右键src文件夹，选择NeW(新建) →Package(软件包)。![[idea_2026-04-02_16-57-50.png]]
3. 输入包名，命名规则为：公司域名的反写+包作 （如com.itheima.test)。![[idea_2026-04-02_17-03-22.png]]
4. 点击OK完成创建。![[idea_2026-04-02_17-04-20.png]]

#### 2.2.3— 新建类
1. 右键刚刚创建的包，选择New(新建)→Java Class(java 类)。![[idea_2026-04-02_17-12-06.png]]
2. 输入类名（如HelloWorld），按回车确认。![[idea_2026-04-02_17-15-22.png]]![[idea_2026-04-02_17-17-47.png]]

#### 2.2.4— 编写代码并运行
在代码编辑区域（右侧）中，编写以下代码：
```java
public class HelloWorld {
	public static void main(String[] args) {
		System.out.println("Hello,World!");
	}
}
```

注意：保持空格、大小写和符号的一致性，所有符号均需在英文输入状态下输入。
如果代码有错，会有红色波浪线提示，需要修改。
#### 2.2.5— 运行程序
1. 右键代码编辑区域，选择RunHelloWorld.main()运行程序。![[idea_2026-04-02_17-40-55.png]]
2. 等待右下角进度条完成后，查着下方控制台的输出结果。![[idea_2026-04-02_17-41-57.png]]
提示：如果想修改输出内容，只需更改代码中的绿色文本部分（双引号内的内容），但双引号本身不能删除。
