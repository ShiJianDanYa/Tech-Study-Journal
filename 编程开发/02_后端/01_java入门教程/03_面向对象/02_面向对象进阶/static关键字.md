特点:叫做静态变量,被该类所有对象共享
调用方式
	1. 类名调用(一般都用 因为这样可以体现静态变量属于类而不是对象)

# 3— 注意事项
1. 静态方法只能访问<font color="#d83931">静态变量</font>和其他<font color="#d83931">静态方法</font>
2. 非静态方法可以访问静态变量或者静态方法,也可以访问非静态的成员变量和非静态成员方法
3. 静态方法中是没有this关键字的
总结:静态只能调用静态,非静态可以调用所有,静态方法无this

# 4— 知识扩展
疑惑为什么静态只能调用静态
![[附件存储/347281457dfe653bcf2e571f36466bfb_MD5.png]]
因为静态是随着类的加载而加载,优先于对象的
而非静态的是和对象有关的,当对象被加载时,静态就已经被加载了,所以静态只能调用静态,非静态可以调用所有.
