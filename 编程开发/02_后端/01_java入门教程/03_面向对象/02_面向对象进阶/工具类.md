# 1— 需要解决问题
方法只能在本类中使用要跨类，就需要在写一次，如何让方法能够跨类使用
# 2— 本质 意义 解决方法
## 1— 工具类
不是用来描述一类事物的，也没有main方法，而是帮我们做一些事情的类
## 2— 书写步骤
1. 类名见名知意
2. 私有化构造方法(非描述事物,创建对象没有意义)
3. 方法定义为静态(无对象,直接类名调用)
