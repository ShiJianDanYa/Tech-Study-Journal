# 1— 需要解决问题
能不能在创建对象,的同时对其属性进行赋值
# 2— 本质 意义 解决方法
## 1— 构造方法
构造方法也叫做构造器,构造函数
作用:在创建对象的同时，给成员变量进行初始化
### 1.1— 格式
```java
修饰符 类名(参数) {
	方法体;
}
//例子
public class Student {
	private String name;
	private int age;
	
	public Student() {
		...空参构造方法
	public Student(String name, int age) {
		...带全部参数构造方法
}

```

构造方法 是 创建对象的时候虚拟机自动调用
由小括号里的参数来决定调用那个构造方法

### 1.2— 构造方法的特点
1. 方法名与类名相同,大小写也要一致
2. 没有返回值,连void都没有
3. 没有具体的返回值(不能由return带回结果数据)
### 1.3— 执行时机
1. 创建对象的时候由虚拟机调用,不能手动调用构造方法
2. 每创建一次对象,就会调用一次构造方法


# 3— 注意事项
## 1— 构造方法注意事项
1. 如果没有定义构造方法,系统将会给出一个<font color="#d83931">默认的无参构造方法</font>
2. 如果自己写了任意构造方法,系统将<font color="#d83931">不再提供</font>默认构造方法
3. 带参构造和无参构造方法,名字相同,参数不同,这叫做构造方法的重载
以上三条都不用记
只需要记住
无是否使用,都手动书写无参构造,和全参构造方法
