# 2— 本质 意义 解决方法
## 1— pivate关键字
是一个权限修饰符,可以修饰成员变量和成员方法
成员变量:定义在方法外面,类里面的变量
![image.png](https://obsidian-1332276919.cos.ap-guangzhou.myqcloud.com/20260310200120010.png)
成员方法 定义在类中行为(方法)
### 1.1— 特点
一旦被private修饰,只能在本类中才能访问,外界无法访问
代码演示
![image.png](https://obsidian-1332276919.cos.ap-guangzhou.myqcloud.com/20260310201439047.png)
此时 age 变量只能在本类中访问,不能直接通过对象操作
好了想在我想赋值该怎么办呢?
我们想要的是 符合标准的可以赋值 不符合的的无法赋值
public 也是一个权限修饰符 意思为公共的 对所有对象开放
直接用public修饰方法,对外提供赋值(输入判断)和获取方法

# 3— 注意事项

成员变量默认初始化值

| 数据类型分类 | 具体类型                        | 默认初始化值                           |
| ------ | --------------------------- | -------------------------------- |
| 基本数据类型 | 整数型（byte/short/int/long）    | 0                                |
|        | 浮点型（float/double）           | 0.0                              |
|        | 字符型（char）                   | `\u0000`（空字符，ASCII 码 0，控制台打印为空白） |
|        | 布尔型（boolean）                | false                            |
| 引用数据类型 | 所有引用类型（String、数组、自定义类、null） | null（表示不指向任何对象）                  |
|        |                             |                                  |
