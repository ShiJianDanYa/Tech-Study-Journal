# 2— 本质 意义 解决方法
## 1— 面向对象
对象:将相关的数据和方法组织为一个整体来看待
面向对象：利用对象进行软件开发
## 2— 类和对象
类：是描述（限定）一类事物
对象：记录单独个体的所有信息
我的理解就是
类：给对象一个统一标准，避免你定义的对象缺胳膊少腿
以狗为列子演示
![image.png](https://obsidian-1332276919.cos.ap-guangzhou.myqcloud.com/20260310105206097.png)
对象：统一标准下的个体
类就是有喉结的就是男生
对象就是具体喉结的大小
对象格式
```java
类名 对象名 = new 类名();
```
例子
Dog d = new Dog( );
new：创建一个大空间
Dog: 管理狗狗的信息
D: 大空间的名字
