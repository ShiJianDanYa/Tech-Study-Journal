## 循环遍历解法
<font color="#6425d0">解法2</font>
<font color="#6425d0">此遍历适用范围是 行高 一样 </font>
<font color="#6425d0">比如 行 高 都为 4  范围 4*4</font>

```java
//解法一  
//遍历一维数组tempArr得到每一个元素,把每一个元素依次添加到二维数组中  
int ar = 0;  
for (int i = 0; i < date.length; i++) {  
    for (int j = 0; j < date[i].length; j++) {  
        date[i][j] = tempArr[ar];
        //自增遍历  
        ++ar;  
  
    }  
  
}  
//解法二  
for (int i = 0; i < tempArr.length; i++) {  
    date[i / 4][i % 4] = tempArr[i];  
}
```
